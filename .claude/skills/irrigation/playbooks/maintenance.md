# Seasonal maintenance

Calendar-driven work: winterize, spring start-up, mid-season checks, filter/diaphragm cadence. Same docs, PDFs, and safety gate as `howto.md`, but framed by the Wijchen, NL climate — real winter freeze risk, mild summer drought, growing season roughly April–October. `setup.yaml`'s laterals note ("zero freeze damage historically") and install dates feed the winterize and inspection logic.

Answer the cadence the user asked about — don't recite the whole calendar for a narrow question. Read the area doc for the specifics, especially `knowledge/laterals.md` for winterizing: this build's dead-end laterals have a documented rule, so use it rather than improvising from generic North-American advice.

Cheat-sheet (a starting point, refine to the ask):
- **Spring start-up (late Mar / early Apr):** confirm the line is dry, crack a bleed screw, bring the pump up slowly, walk each zone 1–2 min watching for unpopped or weak heads, weeping, or wet patches over laterals; check valve-box connectors for moisture; set the season's schedule.
- **Mid-season (monthly, May–Sep):** one full run-through, eyes on every head; check the app log for zones that failed to run; let RainMachine rain-skip/ET handle most schedule adjustment.
- **Winterize (late Oct / early Nov, before hard frost):** follow the `knowledge/laterals.md` dead-end rule; pump off, drain via the manual valve, bleed each zone valve to drain the manifold; re-check any newly exposed sections from this season's work.
- **Annual / multi-year:** clean the filter mesh at spring start-up (replace if torn); inspect the diaphragm every ~5 yr, or sooner if a zone weeps; solenoid o-ring on symptom, not schedule; rotor riser seal if it weeps when off.

Surface images at the steps that need them (filter mesh, bleed screw, leak-inspection points, rotor cap). Route any "...and it was weak last year" toward `irrigation-troubleshoot`.
